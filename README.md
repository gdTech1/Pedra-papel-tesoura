# Pedra, Papel e Tesoura em Python

Um jogo simples de Pedra, Papel e Tesoura feito em Python, que salva o histórico de partidas em um arquivo JSON.

## Funcionalidades
- Jogo contra o computador
- Registro de vitórias, derrotas e empates
- Histórico salvo em arquivo JSON
- Visualização do histórico

## Tecnologias usadas
- Python 3
- JSON (armazenamento de dados)
- Random (biblioteca python)
- Datetime (biblioteca python)
